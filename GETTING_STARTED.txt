================================================================================
  FIRST TIME USING THE APP?
================================================================================

Follow these steps to test all features:

1. CREATE PROFILES
   - Click "Select current profile..." (top right)
   - Type "John" and click "Add"
   - Create "Sarah" and "Alex" the same way

2. CREATE AN EVENT
   - Select John from dropdown
   - Check "John" in the profiles list
   - Select timezone: Eastern Time (ET)
   - Pick tomorrow's date
   - Set time: 10:00 AM to 11:00 AM
   - Click "Create Event"

3. TEST TIMEZONE CONVERSION
   - Change "View in Timezone" to Pacific Time (PT)
   - Watch the time change from 10:00 AM to 7:00 AM
   - Try India (IST) - it becomes 7:30 PM!

4. UPDATE AN EVENT
   - Click on the event card
   - Change the start time to 2:00 PM
   - Click "Update Event"

5. VIEW UPDATE HISTORY
   - Click "View Logs" on the event
   - See the change from 10:00 AM to 2:00 PM
   - Notice the timestamp

================================================================================
  API ENDPOINTS
================================================================================

Profiles:
  GET    /api/profiles                    - Get all profiles
  POST   /api/profiles                    - Create profile
  PATCH  /api/profiles/:id/timezone       - Update timezone

Events:
  GET    /api/events/profile/:profileId   - Get events for profile
  GET    /api/events/:id                  - Get single event
  POST   /api/events                      - Create event
  PATCH  /api/events/:id                  - Update event
  DELETE /api/events/:id                  - Delete event

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "MongoDB connection error"
Solution: Make sure MongoDB is running
         $ mongod

Problem: "Port 5000 already in use"
Solution: Kill the process
         $ lsof -ti:5000 | xargs kill -9

Problem: "Cannot find module..."
Solution: Reinstall dependencies
         $ rm -rf node_modules && npm install

Problem: Frontend won't start
Solution: Clear cache and reinstall
         $ cd client
         $ rm -rf node_modules package-lock.json
         $ npm install
